# Workspace Folder

This folder is used for all generated content and temporary work files created by Claude Code skills and commands.

## Purpose

- Generated PRDs (HTML, Markdown)
- Analysis reports
- Exported data
- Temporary files created during skill execution
- Any other generated artifacts

## Structure

You can organize this folder however you like. Common subfolders:
- `prds/` - Product Requirements Documents
- `reports/` - Analysis reports
- `exports/` - Exported data from integrations

## Note

This folder's contents are gitignored (except this README). Files here are local to your machine and won't be committed to version control.
